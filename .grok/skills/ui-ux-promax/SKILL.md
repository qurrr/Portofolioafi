# UI/UX ProMax

## When to use this skill
- Saat user ingin improve tampilan website (terutama portfolio)
- Membuat atau memperbaiki design system
- Menambahkan animasi, hover effect, responsive design
- Membuat UI yang modern, clean, dan premium

## Core Capabilities
- Expert di Tailwind CSS + custom design system
- Modern trends 2026 (glassmorphism, neumorphism, 3D hover, scroll animations)
- Accessibility (WCAG) & SEO friendly
- Dark/Light mode toggle
- Mobile-first responsive
- Micro-interactions & smooth UX

## Best Practices yang selalu dipakai
- Gunakan semantic HTML + Tailwind
- Hero section yang impactful
- Smooth scroll + lenis atau locomotive scroll
- Card component yang elegan
- Navigation yang sticky & responsive
- Portfolio grid dengan filter & modal
- Contact form yang cantik
- Footer yang premium

## Prompting Guide
Gunakan prompt seperti:
- "Improve UI/UX section hero dengan style modern"
- "Buat component portfolio card dengan glassmorphism effect"
- "Tambahkan dark mode toggle yang smooth"
- "Make this website look premium and professional"
